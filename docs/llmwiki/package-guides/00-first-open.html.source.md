## First Step Cards

### Step 1: 起動

Windows の人は、`Windows用` を開きます。
その中の `動画保存ツール.exe` を開きます。

Mac の人は、`Mac用` を開きます。
その中の `動画保存ツール.app` を開きます。

いつもと違う警告が出たときは、無理に進めないでください。
先に「困ったとき」を見てください。

### Step 2: URL貼り付け

アプリが開いたら、保存したいページの URL を貼り付けます。
手入力でもかまいません。

使えるのは、あなたが保存してよいページだけです。

### Step 3: 保存

保存ボタンを押します。
進み具合が出たら、終わるまで待ちます。

保存中は、アプリを閉じないでください。

### Step 4: 保存先を開く

保存が終わったら、アプリ内の「保存先を開く」を押します。
保存したファイルを確認できます。

保存先が見つからないときも、まず「保存先を開く」を使ってください。

### Step 5: 停止して終了

終わるときは、アプリ内の「停止して終了」を使います。

右上の X で閉じても、裏側の保存処理が止まらない場合があります。
将来の専用アプリでは、閉じる前に確認が出る予定です。

## Warning Box

このツールはローカル専用です。
Web 公開、広告、外部の人向けサービスには使いません。

Cookie、token、secret を人に送らないでください。
この案内では、それらの値を入力したり、共有したりしません。

DRM 回避、ログイン回避、制限回避には使いません。
保存してよい内容だけに使ってください。

## In-App Help Cards

### 使い方

起動、URL貼り付け、保存、保存先を開く、停止して終了の順に進めます。
迷ったら、この順番に戻ります。

### 困ったとき

起動しない、保存できない、保存先がわからない、終了方法がわからない。
そのときは、まずアプリ内の「困ったとき」を見ます。

### 安全な使い方

保存してよい内容だけを扱います。
公開、広告、共有、回避のためには使いません。

### 保存先を開く

保存したファイルを探すときは、アプリ内の「保存先を開く」を使います。
別の場所を探し回る前に、ここを押してください。

### 停止して終了

作業が終わったら、アプリ内の「停止して終了」を使います。
保存中なら、終わるまで待ってから終了します。

## Trouble Cards

困ったときは、まずアプリ内の「困ったとき」を見ます。
そこに同じ内容があれば、その案内を先に使います。

### 起動しない

もう一度だけ開きます。
それでも起動しないときは、アプリ内の「困ったとき」を見ます。
警告の画面が出ている場合は、無理に進めません。

### 保存先不明

アプリ内の「保存先を開く」を押します。
それでも見つからないときは、「困ったとき」を見ます。

### 保存できない

URL が正しいかを見ます。
そのページを保存してよいかも確認します。
何度も失敗するときは、エラー文を残して「困ったとき」を見ます。

### 終了方法不明

アプリ内の「停止して終了」を使います。
右上の X だけで閉じるのは避けます。
