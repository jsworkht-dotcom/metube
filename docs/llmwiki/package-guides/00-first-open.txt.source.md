## TXT Body

動画保存ツール_ローカル専用
最初に読んでください

1. これは何か

このツールは、自分の PC 内だけで使う動画保存ツールです。
Web 公開や外部共有のためのものではありません。

2. 使い方

1. アプリを起動します。
2. 保存したいページの URL を貼り付けます。
3. 保存します。
4. 保存が終わったら、「保存先を開く」を押します。
5. 使い終わったら、「停止して終了」を押します。

保存中は、終わるまで待ってください。
右上の X だけで閉じると、裏側の処理が止まらない場合があります。

3. 安全な使い方

自分の動画、許可済み動画、合法利用できる動画だけに使います。
Web 公開しません。
広告収益化しません。
cookie/token/secret を共有しません。
DRM 回避、認証突破、制限回避には使いません。

4. 困ったとき

まず画面内の「困ったとき」を見ます。
保存先が分からないときは、「保存先を開く」を押します。
終了方法が分からないときは、「停止して終了」を押します。

5. 見やすい説明

見やすい説明は、00_最初に開いてください.html を開いてください。
