## Quick Steps

1. アプリを起動します。
2. URL を貼り付けます。
3. 保存形式を選びます。
4. 保存します。
5. 保存先を開きます。
6. 停止して終了します。

## Action Cards

### 1. アプリを起動する

アプリを開きます。
画面が出るまで少し待ちます。

画面が出ないときは、無理に何度も開かず、「困ったとき」を見ます。

### 2. URL を貼り付ける

保存したいページの URL を入力欄に貼り付けます。
手入力したときは、文字が抜けていないかを見ます。

保存してよいページだけを使ってください。

### 3. 保存形式を選ぶ

まずは、かんたん設定を使います。
分からないときは、標準の設定を選びます。

詳細設定は、意味が分かる人だけ使います。

### 4. 保存する

保存ボタンを押します。
保存中の表示が出たら、終わるまで待ちます。

保存中は、アプリを閉じないでください。

### 5. 保存先を開く

保存が終わったら、「保存先を開く」を押します。
保存したファイルを確認できます。

ファイルが見つからないときも、まず「保存先を開く」を押します。

### 6. 停止して終了する

使い終わったら、「停止して終了」を押します。
右上の X だけで閉じると、裏側の処理が止まらない場合があります。

保存中に終了したいときは、画面の確認を読んでから選びます。

## Format Cards

### 動画として保存

映像と音声をまとめて保存したいときに使います。
迷ったときは、この形式を選びます。

### 音声だけ保存

音だけを残したいときに使います。
映像は保存しません。

### 標準

ふだん使う設定です。
迷ったときは、標準を選びます。

### 高画質

きれいに保存したいときに使います。
保存に時間がかかる場合があります。

### かんたん設定

はじめて使うときは、かんたん設定を優先します。
細かい設定を知らなくても保存できます。

### 詳細設定

詳細設定は、意味が分かる人だけ使います。
分からないときは、変更しません。

## Status Explanation Cards

### 保存中

保存が進んでいます。
終わるまで待ちます。

### 完了

保存が終わりました。
「保存先を開く」を押して、ファイルを確認します。

### 失敗した場合

URL が正しいかを見ます。
そのページを保存してよいかも確認します。

### もう一度試す

エラー文を残します。
同じ URL をもう一度だけ試します。
何度も失敗するときは、「困ったとき」を見ます。

## Save Folder Guidance

保存後は、「保存先を開く」を押します。
ファイルが見つからないときも、ここを見ます。

保存先を自分で探し回る前に、画面の「保存先を開く」を使ってください。

## Stop And Quit

使い終わったら、「停止して終了」を押します。
保存中は、終了前に確認します。

右上の X だけでは、裏側の処理が止まらない場合があります。

## Warning Box

このツールはローカル専用です。
自分の動画、許可済み動画、合法利用できる動画だけに使います。

Web 公開しません。
広告収益化しません。
cookie/token/secret を共有しません。
DRM 回避、認証突破、制限回避には使いません。

## Troubleshooting Link Cards

### 起動しない

まず画面内の「困ったとき」を見ます。
いつもと違う警告が出たときは、無理に進めません。

### 保存できない

URL と保存してよいページかを確認します。
同じエラーが続くときは、「困ったとき」を見ます。

### 保存先が分からない

「保存先を開く」を押します。
それでも分からないときは、「困ったとき」を見ます。

### 画面が固まったように見える

保存中の表示があるかを見ます。
すぐに閉じず、しばらく待ちます。
分からないときは、「困ったとき」を見ます。

## Footer Note

基本は、URL を貼り付けて、保存して、保存先を開きます。
終わったら「停止して終了」を押します。

安全な使い方を確認したいときは、画面内の「安全な使い方」を見てください。
