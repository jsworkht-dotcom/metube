## TXT Body

使い方
動画を保存する基本手順

1. 基本手順

1. アプリを起動します。
2. URL を貼り付けます。
3. 保存形式を選びます。
4. 保存します。
5. 保存先を開きます。
6. 停止して終了します。

2. 保存形式

動画として保存できます。
音声だけ保存することもできます。
標準と高画質を選べます。
迷ったら標準を選びます。

3. 保存中と完了

保存中は、終わるまで待ちます。
完了したら、「保存先を開く」を押します。
失敗したら、もう一度だけ試します。
何度も失敗するときは、「困ったとき」を見ます。

4. 終了方法

使い終わったら、「停止して終了」を押します。
右上の X だけで閉じると、裏側の処理が止まらない場合があります。

5. 安全な使い方

自分の動画、許可済み動画、合法利用できる動画だけに使います。
Web 公開しません。
広告収益化しません。
cookie/token/secret を共有しません。
DRM 回避、認証突破、制限回避には使いません。

6. 見やすい説明

見やすい説明は、03_使い方.html を開いてください。
