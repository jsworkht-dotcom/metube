## TXT Body

困ったとき
うまく動かないときの確認

1. まず試すこと

1. アプリを一度終了して、もう一度起動します。
2. 保存中なら、少し待ってからもう一度試します。
3. 「保存先を開く」を押して確認します。
4. 画面内の状態表示を見ます。
5. 表示された文は、消さずに残します。

2. よくある困りごと

アプリが開かない
少し待って、もう一度だけ起動します。
いつもと違う警告が出たときは、無理に進めません。

URLを入れても保存できない
URL が抜けていないかを見ます。
保存してよいページか確認します。
同じ URL をもう一度だけ試します。

保存先が分からない
「保存先を開く」を押します。
保存が完了しているか、状態表示も見ます。

保存が途中で止まったように見える
保存中なら、少し待ちます。
失敗の表示が出たら、もう一度だけ試します。

保存したファイルが見つからない
「保存先を開く」を押します。
最近保存したファイルを見ます。

終了方法が分からない
使い終わったら、「停止して終了」を押します。

更新表示が分からない
表示された文を読みます。
分からないときは、その場で進めません。

3. 終了方法

使い終わったら、「停止して終了」を押します。
右上の X だけで閉じると、裏側の処理が止まらない場合があります。
保存中なら、終わるまで待ってから終了します。

4. 安全な使い方

自分の動画、許可済み動画、合法利用できる動画だけに使います。
Web 公開しません。
広告収益化しません。
cookie/token/secret を共有しません。
DRM 回避、認証突破、制限回避には使いません。

5. 見やすい説明

見やすい説明は、04_困ったとき.html を開いてください。
