## Hero

### Title

安全な使い方

### Heading

このツールを安心して使うために

### Lead

このツールは、自分の PC 内だけで使うローカル専用の動画保存ツールです。
保存してよい内容だけを、自分の学習、整理、バックアップのために扱います。

## Safe-Use Cards

### 自分のPC内で使う

保存したファイルは、自分の PC 内で確認します。
外に公開するための道具ではありません。

### ローカル専用

このツールはローカル専用です。
他の人が外から使えるサービスにはしません。

### 個人利用

自分の学習、整理、バックアップのために使います。
外部ユーザー向けのサービスにはしません。

## Do Cards

### 自分がアップロードした動画

自分がアップロードした動画を、自分の確認や整理のために保存します。

### 権利者から許可を得た動画

保存してよいと分かっている動画だけを扱います。

### 合法的に保存できる素材

保存してよい条件がはっきりしている素材だけを使います。

### 自分の学習・整理・バックアップ用途

見返す、整理する、控えを残すなど、自分のための用途にします。

## Do-Not Cards

### Web公開しない

このツールを Web に公開しません。
外から使えるようにしません。

### 広告収益化しない

保存した内容やこのツールを、広告収益化のために使いません。

### 外部ユーザーに使わせない

他の人向けの保存サービスにしません。

### DRM回避に使わない

保護を外す目的では使いません。

### 認証突破に使わない

ログインや許可が必要な場所を無理に通る目的では使いません。

### 制限回避に使わない

保存できない制限を無理に避ける目的では使いません。

### 大量取得目的にしない

大量に集めることを目的にしません。

## Sensitive-Data Warning Box

次のものは共有しません。
人に送らず、画面にも貼り付けません。

- cookie
- token
- secret
- `.env`
- cookies.txt
- 個人の保存データ
- 個人バックアップ

この案内では、これらの中身を入力したり、表示したり、保存したりしません。

## Safe Trouble Actions

分からない URL は使いません。
保存できない場合は、無理に回避しません。
画面内の「困ったとき」を見ます。
終了するときは、「停止して終了」を使います。

## Update-Safety Note

更新状態は画面に表示されます。
このツールは勝手に更新しません。
更新する場合は、バックアップと確認が必要です。

現段階では、自動更新実行はしません。
分からない更新表示が出たときは、その場で進めず「困ったとき」を見ます。

## Footer Note

安心して使うための基本は、保存してよい内容だけを、自分の PC 内で扱うことです。
迷ったときは、保存せずに一度止まり、「困ったとき」と「安全な使い方」を確認します。
